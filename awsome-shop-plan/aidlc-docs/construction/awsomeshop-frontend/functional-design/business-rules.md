# Unit 1: awsomeshop-frontend — 业务规则

---

## 1. 表单校验规则

### 1.1 注册表单

| 字段 | 规则 | 错误提示 |
|------|------|---------|
| 用户名 | 必填 | 请输入用户名 |
| 用户名 | 3-20位字符 | 用户名长度为3-20位 |
| 用户名 | 仅字母、数字、下划线 | 用户名只能包含字母、数字和下划线 |
| 密码 | 必填 | 请输入密码 |
| 密码 | 最少6位 | 密码长度至少6位 |

### 1.2 登录表单

| 字段 | 规则 | 错误提示 |
|------|------|---------|
| 用户名 | 必填 | 请输入用户名 |
| 密码 | 必填 | 请输入密码 |

### 1.3 产品表单（管理端）

| 字段 | 规则 | 错误提示 |
|------|------|---------|
| 产品名称 | 必填 | 请输入产品名称 |
| 产品名称 | 最多100字符 | 产品名称不能超过100字符 |
| 所需积分 | 必填 | 请输入所需积分 |
| 所需积分 | 正整数 | 所需积分必须为正整数 |
| 库存数量 | 必填 | 请输入库存数量 |
| 库存数量 | 非负整数 | 库存数量不能为负数 |
| 产品分类 | 必填 | 请选择产品分类 |
| 产品图片 | 文件类型 jpg/png/gif | 仅支持 jpg、png、gif 格式 |
| 产品图片 | 文件大小 ≤ 5MB | 图片大小不能超过 5MB |

### 1.4 分类表单（管理端）

| 字段 | 规则 | 错误提示 |
|------|------|---------|
| 分类名称 | 必填 | 请输入分类名称 |
| 分类名称 | 最多50字符 | 分类名称不能超过50字符 |
| 父分类 | 最多2级 | 分类层级不能超过2级 |

### 1.5 积分调整表单（管理端）

| 字段 | 规则 | 错误提示 |
|------|------|---------|
| 调整金额 | 必填 | 请输入调整金额 |
| 调整金额 | 整数（正负均可） | 调整金额必须为整数 |
| 调整金额 | 扣减后余额 ≥ 0 | 扣减后积分不能为负数 |
| 调整原因 | 必填 | 请输入调整原因 |
| 调整原因 | 最多200字符 | 调整原因不能超过200字符 |

### 1.6 积分配置表单（管理端）

| 字段 | 规则 | 错误提示 |
|------|------|---------|
| 每月发放积分 | 必填 | 请输入每月发放积分数量 |
| 每月发放积分 | 正整数 | 发放积分必须为正整数 |

## 2. 权限控制规则

### 2.1 路由权限

| 路由类型 | 访问条件 | 未满足时行为 |
|---------|---------|-------------|
| 公开路由 | 无 | 允许访问 |
| 公开路由 | 已登录用户 | 重定向到首页 |
| 员工路由 | 已登录 | 未登录重定向到登录页 |
| 管理员路由 | 已登录 + 管理员角色 | 未登录重定向到登录页，非管理员重定向到首页 |

### 2.2 功能权限

| 功能 | 权限要求 |
|------|---------|
| 浏览产品 | 已登录 |
| 查看积分 | 已登录 |
| 发起兑换 | 已登录 |
| 查看兑换历史 | 已登录 |
| 产品管理 | 管理员 |
| 分类管理 | 管理员 |
| 积分管理 | 管理员 |
| 兑换记录管理 | 管理员 |
| 用户管理 | 管理员 |

## 3. 交互规则

### 3.1 兑换规则

| 规则 | 说明 |
|------|------|
| 积分校验 | 兑换前检查当前积分 ≥ 所需积分 |
| 库存校验 | 由后端校验，前端显示库存供参考 |
| 二次确认 | 跳转确认页，显示详细信息后确认 |
| 兑换成功 | 显示成功提示，跳转兑换历史页 |
| 兑换失败 | 显示具体错误原因 |

### 3.2 分页规则

| 场景 | 分页方式 | 每页数量 |
|------|---------|---------|
| 员工端产品列表 | 无限滚动 | 20 |
| 员工端兑换历史 | 无限滚动 | 20 |
| 员工端积分历史 | 无限滚动 | 20 |
| 管理端产品列表 | 传统分页 | 10 |
| 管理端兑换记录 | 传统分页 | 10 |
| 管理端积分列表 | 传统分页 | 10 |
| 管理端用户列表 | 传统分页 | 10 |

### 3.3 搜索规则

| 规则 | 说明 |
|------|------|
| 防抖 | 输入后 300ms 触发搜索 |
| 最小长度 | 无限制（空字符串显示全部） |
| 搜索范围 | 产品名称 |
| 结果展示 | 重置分页，显示搜索结果 |

|--------|---------|---------|

## 5. 错误处理规则

### 5.1 HTTP 状态码处理

| 状态码 | 处理方式 |
|--------|---------|
| 200 | 正常处理响应 |
| 400 | 显示业务错误信息 |
| 401 | 清除认证状态，跳转登录页 |
| 403 | 显示"权限不足" |
| 404 | 显示"资源不存在" |
| 500 | 显示"服务器错误，请稍后重试" |
| 502 | 显示"服务暂时不可用" |
| 504 | 显示"请求超时，请重试" |

### 5.2 业务错误码处理

| 错误码前缀 | 来源 | 处理方式 |
|-----------|------|---------|
| AUTH_ | auth-service | 显示认证相关错误 |
| PROD_ | product-service | 显示产品相关错误 |
| PTS_ | points-service | 显示积分相关错误 |
| ORD_ | order-service | 显示兑换相关错误 |
| GW_ | api-gateway | 显示网关错误 |

### 5.3 常见错误提示

| 错误码 | 提示文本 |
|--------|---------|
| AUTH_001 | 用户名或密码错误 |
| AUTH_002 | 用户名已存在 |
| AUTH_003 | 账号已被禁用 |
| PROD_001 | 产品不存在 |
| PROD_002 | 库存不足 |
| PTS_001 | 积分不足 |
| ORD_001 | 订单不存在 |
| GW_001 | 登录已过期，请重新登录 |
| GW_002 | 权限不足 |

## 6. 图片处理规则

### 6.1 图片懒加载

| 规则 | 说明 |
|------|------|
| 触发时机 | 图片进入视口时加载 |
| 占位显示 | 加载前显示灰色占位 |
| 加载失败 | 显示默认占位图 |

### 6.2 图片上传

| 规则 | 说明 |
|------|------|
| 支持格式 | jpg, png, gif |
| 大小限制 | 5MB |
| 上传方式 | multipart/form-data |
| 预览 | 上传成功后显示预览 |

## 7. 本地存储规则

### 7.1 存储项

| 键名 | 类型 | 生命周期 |
|------|------|---------|
| token | string | 登录后存储，退出时清除 |
| user | JSON | 登录后存储，退出时清除 |

### 7.2 存储操作

| 操作 | 时机 |
|------|------|
| 写入 | 登录成功后 |
| 读取 | 应用启动时 |
| 清除 | 退出登录时、401 响应时 |

## 9. 边界条件

### 9.1 空状态处理

| 场景 | 显示内容 |
|------|---------|
| 产品列表为空 | "暂无产品" |
| 搜索无结果 | "未找到相关产品" |
| 兑换历史为空 | "暂无兑换记录" |
| 积分历史为空 | "暂无积分变动记录" |

### 9.2 加载状态处理

| 场景 | 显示内容 |
|------|---------|
| 页面加载中 | 全屏加载指示器 |
| 列表加载中 | 列表底部加载指示器 |
| 按钮操作中 | 按钮禁用 + 加载图标 |
| 图片加载中 | 灰色占位 |

### 9.3 数值边界

| 场景 | 边界 |
|------|------|
| 积分余额 | ≥ 0 |
| 产品库存 | ≥ 0 |
| 所需积分 | > 0 |
| 分页页码 | ≥ 0 |
| 每页数量 | 10 或 20 |
