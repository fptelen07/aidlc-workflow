## Unit 6: API 网关 (api-gateway)

| 故事 | 描述 | 优先级 |
|------|------|--------|
| US-025 | 管理员登录（网关层角色权限校验） | Must Have |

API 网关作为横切关注点，间接支撑所有需要认证和权限校验的用户故事：
- 所有员工端故事（US-004 ~ US-012）：网关负责 JWT 令牌校验
- 所有管理员端故事（US-013 ~ US-024）：网关负责管理员角色校验
- 公开端点放行（US-001 注册、US-002 登录）：网关配置白名单
